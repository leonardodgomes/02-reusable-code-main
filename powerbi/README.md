# 📊 Power BI Portfolio

Welcome to my **Power BI Portfolio**, a curated collection of dashboards, reusable DAX patterns, templates, and complete analytics projects.
This repository is designed to showcase my approach to data modeling, visualization, and analytical problem‑solving using Microsoft Power BI.

It is organized into four main sections:

- DAX Library — advanced, reusable DAX patterns
- Templates — themes, layouts, and report structures
- Projects — complete end‑to‑end Power BI solutions
- Assets — icons, backgrounds, and shared resources

Each folder is structured to be clean, scalable, and easy to navigate.

## 📘 DAX Library
The DAX folder contains a complete library of advanced DAX logic, organized into:

### 📅 DateTable
Reusable calendar tables:

- Full calendar
- Fiscal calendar
- Weekly calendar
- Holiday logic

### ⏱️ Time Intelligence
Professional time‑based calculations:

- YTD, MTD, QTD
- Rolling windows
- Same period last year
- Moving averages
- Leap‑year‑safe logic

### 🧩 Patterns
Reusable analytical techniques:

- Dynamic ranking
- TopN with “Others”
- Segmentation
- KPI status logic
- Percentile bands
- Virtual tables
- Dynamic titles

### 🔗 Relationship Techniques
Advanced modeling logic:

- USERELATIONSHIP
- CROSSFILTER
- Many‑to‑many solutions
- Disconnected tables
